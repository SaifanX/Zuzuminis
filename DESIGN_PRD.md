# Zuzu Minis: Design PRD (Visual Blueprint)

## 1. Aesthetic Identity: "Playful Premium"
*   **Vibe**: High-end boutique warmth, minimalist luxury, lots of whitespace.
*   **Color Palette**:
    *   `Zuzu Blue`: Teal / Soft Blue (#4A90E2)
    *   `Minis Pink`: Soft Pink / Peach (#FFB6C1)
    *   `Base`: Cream / Off-White (#FDFCF9)
    *   `Text`: Deep Charcoal / Slate (#1A1A1A)
*   **Typography**:
    *   `Headings`: **Searghy** (Playful, high-character display serif).
    *   `Body/Accents`: **Kiddos** (Friendly, rounded kid-like font).

## 2. Core Visual Strategy
*   **Editorial Layout**: Use massive, asymmetrical typography and large imagery to create an "Elite" feel.
*   **Weightlessness**: Elements should feel floating and responsive (GSAP parallax).
*   **The Mascot**: A high-fidelity, 3D version of the "Zuzu" baby face that reacts to mouse movement.

## 3. Site Architecture (Key Chapters)
1.  **Chapter 01: The Statement**: Hero section with bold branding and the interactive mascot.
2.  **Chapter 02: The Collection**: Cinematic product showcase (Horizontal scroll or editorial grid).
3.  **Chapter 03: The Story**: Narrative section on the Bangalore studio and fabric quality.
4.  **Chapter 04: The Shop**: High-performance, custom-built product index and checkout flow.
